Our Mission

Simplicity

TensowFlow UI is built to be as straightforward as possible. No nonsense cluttering up your workspace, and no complex, obscure features: we want all users to feel comfortable and at-home right from the moment they start using the app.

Accessibility

In much the same way, we want anybody to be able to use our program - from beginners to experts alike. If you want to try it, don't be afraid - give it a go, and see where it takes you!

Utility

What good is a program without functionality? None at all - that's why we want maximum functionality to go along with ease of use, allowing everyone to make useful ML models when they try our system.

Features

Visual interface

As the UI for TensorFlow, the interface is of vital importance. We wanted everything to be visually represented in some way, with no need for the user to ever examine code should they not wish to.

Simplified experience

The experience is flowing and intuitive, allowing users to easily find their way around, and letting them explore to their heart's content without losing understanding of what they're doing.

Code generation

You can easily generate TensorFlow code from your visual models, allowing you to experience the framework just as though you wrote the code yourself, with minimal hassle and effort on your side.

External output

The code can even be run on our servers, allowing you to see the outputs of the project you've created!

Community library

Projects can be stored in the cloud, on a community library with universal access, allowing you to either import projects others have made, or export your own projects for others to enjoy!
